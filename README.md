# Quiz Game (CLI)

## Overview
This is a simple **Python command-line quiz game**. Users answer multiple-choice questions and get a score at the end.

## Features
- Multiple-choice questions
- Keeps track of score
- Case-insensitive answer checking

## How to Run
1. Open `quiz_game.py` in a Python environment that supports input and print (e.g., Pyto, Carnets, or any desktop Python IDE).  
2. Run the script:  
```bash
python quiz_game.py
```

## Example Interaction                
```
What is the capital of France?
Your answer: Paris
Correct!

Which planet is known as the Red Planet?
Your answer: Mars
Correct!

What is 5 + 7?
Your answer: 10
Wrong! Correct answer: 12

Who wrote Hamlet?
Your answer: Shakespeare
Correct!

What is the largest ocean?
Your answer: Atlantic
Wrong! Correct answer: Pacific

Your final score is 3 out of 5
```
## Skills Demonstrated                  
   - Python fundamentals: loops, dictionaries, conditionals
   - Handling user input and validation
   - CLI-based interactive application
   - Tracking and displaying user score
